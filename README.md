# 🚗 Indian Road Traffic Accident Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes Indian road traffic accident data to identify accident trends, fatalities, injuries, state-wise hotspots, and monthly patterns. The dashboard helps understand road safety issues and supports data-driven decision-making.

## 🎯 Objectives
- Analyze total accidents, fatalities, and injuries
- Identify accident-prone states
- Monitor monthly accident trends
- Compare accident severity across regions

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📊 KPIs
- Total Accidents
- Total Fatalities
- Total Injuries
- Accident Severity Rate

## 📈 Dashboard Features
- KPI Cards
- State-wise Map
- Monthly Trend Analysis
- Vehicle Type Analysis
- Interactive Filters and Slicers

## 💡 Key Insights
- Identified states with the highest accident rates.
- Analyzed seasonal accident patterns.
- Compared fatalities and injuries across regions.
- Highlighted trends to support road safety planning.
